# 🚀 Code Stream (Вайбовий код-шерінг для викладачів)

Ця утиліта дозволяє в один клік розшарити папку з кодом для учнів у локальній мережі. Ніяких складних налаштувань веб-серверів — просто перетягни папку на `.exe` файл!

**Фішки:**
* 🌳 Дерево папок (як у VS Code).
* ✨ Підсвітка синтаксису (Prism.js).
* 🟡 Автоматична підсвітка рядків, які змінилися за останні 10 хвилин.
* 🔄 Автооновлення сторінки при збереженні файлу (без перезавантаження).
* 🪄 Магічні локальні домени (Bonjour/mDNS) — жодних нудних IP-адрес.
* 🖼️ Підтримка перегляду зображень та кнопка завантаження файлів.

---

## 🛠 Як налаштувати та зібрати проект

Для початку переконайся, що в тебе встановлений **Python 3.x**.

### Крок 1. Створення віртуального середовища (venv)
Відкрий термінал у папці з проектом (там, де лежить `server.py`) і виконай команду:

```bash
# Створюємо віртуальне середовище з назвою 'venv'
python -m venv venv
```

Тепер його потрібно активувати:
* **Для Windows:** `venv\Scripts\activate`
* **Для macOS/Linux:** `source venv/bin/activate`

*(Якщо все пройшло успішно, у терміналі зліва з'явиться префікс `(venv)`).*

### Крок 2. Встановлення залежностей
Встанови необхідні бібліотеки через pip:

```bash
pip install flask zeroconf pyinstaller
```

### Крок 3. Збірка в один EXE-файл
Коли всі бібліотеки встановлені, запускаємо PyInstaller, щоб спакувати все в один портативний екзешник:

```bash
pyinstaller --onefile --console --icon=icon.ico codeStream.py
```
* `--onefile` — пакує весь проект в один `.exe` файл.
* `--console` — залишає вікно терміналу (корисно, щоб бачити, що сервер працює, і закрити його хрестиком).

## 🎮 Як користуватися

Цей додаток має круту фішку "Zero-config". Тобі не потрібно нічого прописувати в коді чи конфігах, усе керується назвою самого `.exe` файлу та Drag & Drop.

### 1. Налаштування доменного імені
**Приклади перейменування файлу:**
* `code_stream (code).exe` ➡️ Учні заходять на **http://code.local**

*(Якщо дужок немає, домен буде створено з самої назви файлу, наприклад `server.exe` -> `server.local`)*

### 2. Запуск сервера
1. Візьми папку з поточним уроком (наприклад, `Lesson_5_PyGame`).
2. **Перетягни (Drag & Drop)** цю папку прямо на іконку твого перейменованого `.exe` файлу.
3. Відкриється чорне вікно терміналу, яке напише, за яким посиланням доступний код.

### 3. Що роблять учні
Ти просто пишеш на дошці посиланн. Учні відкривають його в браузері і бачать твій код. 

Як тільки ти натискаєш `Ctrl+S` у своєму редакторі, код у браузері учнів оновиться автоматично за 2 секунди, а змінені рядки підсвітяться зеленим!

---
*Примітка для Windows: Якщо у когось з учнів не відкривається посилання `.local`, вони завжди можуть використати IP-адресу, яка також виводиться в шапці сайту.*
